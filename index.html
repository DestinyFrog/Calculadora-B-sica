Calculadora Básica

Use espaço entre os números e simbolos (a + b - c)
mantenha as operações mais importantes na frente
{3/ ; 2/ ; ^ [ * ; / ( + ; - )]}

Soma => | a + b |
Subtração => | a - b |
Mutiplicação => | a * b |
Divisão => | a / b |
Potenciação => | a ^ b |
Raiz quadrada => | 2/ a |
Raiz cúbica => | 3/ a |
